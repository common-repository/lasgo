== Description ==

Lasgo is an interactive learn-as-you-go WordPress tutorial plugin that helps you learn the basics of WP by displaying tips about the page you select, searching for answers or taking an exam to test your skills.
In the plugin are included jokes as well, which Lasgo will display from time to time. If you find them annoying, you can disable them in the Lasgo Settings area.

= Features =

-Interactive interface

-Search most popular tutorials websites

-Post a question (community oriented)

-Earning points for different tasks

-Exams for 7 Levels

-Learn more section on the most important WP admin pages


Brought to you by <a href="http://wpicode.com/">Wpicode</a>

== Installation ==

Installation is simple:

1. Download the Lasgo plugin
2. Upload with Wordpress Plugins
3. Activate
4. Go in your WP admin area, Lasgo should popup
